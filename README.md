enchant.jsの講習用プログラムです

以下のファイルに分かれています

|_ session1
|_ session1_for_editing

session1 には完成形のプログラムが入っています。
session1_for_editting には編集用のプログラムが入っています。講習ではこちらに書き込んでいって下さい。

それぞれのディレクトリの中には

|_ part1	ゲームの構成要素
|_ part2　ラベル
|_ part3　画像表示
|_ part4　タッチ操作
|_ part5　フレーム操作
|_ part6　移動
|_ part7　矢印キーの入力受け付け
|_ part8　クラスの作成
|_ part9　まとめ

という構成になっています。
